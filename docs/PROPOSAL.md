## 1. 프로젝트 개요

### 1.1 프로젝트명
**의료광고 AI 사전 심의 시스템**

### 1.2 프로젝트 목표
의료광고 심의 과정에서 발생하는 인적 리소스를 절감하고 심의 반려율을 낮추기 위해, 광고 이미지 내 텍스트를 자동으로 추출하고 의료법 및 심의 가이드라인과 대조하여 **통과 가능성을 예측하는 AI 기반 시스템**을 구축합니다.

### 1.3 주요 기능
| 기능 | 설명 |
|------|------|
| OCR 텍스트 추출 | 광고 이미지에서 텍스트 자동 추출 |
| 위반 키워드 탐지 | 의료법 금지 표현 자동 검출 |
| AI 법적 근거 생성 | GPT 기반 위반 사유 및 수정 권고 |
| RAG 법규 검색 | 최신 의료법/가이드라인 기반 분석 |
| 위험도 스코어링 | 0-100점 기반 통과/반려 예측 |

## 2. 제안 배경 및 필요성

### 2.1 현황 및 문제점

#### 의료광고 심의 현황
- 연간 의료광고 심의 건수: **약 10만 건 이상**
- 평균 심의 소요 시간: **3-5일**
- 1차 심의 반려율: **약 30-40%**

#### 주요 문제점
| 문제 | 영향 |
|------|------|
| 인력 의존적 심의 | 심의관 피로도 증가, 일관성 저하 |
| 반복적인 위반 사항 | 동일 유형 반려 반복 발생 |
| 긴 심의 대기 시간 | 광고 집행 지연, 기회 비용 발생 |
| 법규 변경 대응 어려움 | 최신 가이드라인 반영 지연 |

### 2.2 AI 도입 필요성

```
┌─────────────────────────────────────────────────────────────┐
│                    AS-IS (현재)                              │
│  광고주 → 광고 제출 → 심의관 검토 → 반려 → 수정 → 재제출    │
│                         (3-5일)      (30-40%)                │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                    TO-BE (AI 도입 후)                        │
│  광고주 → AI 사전 검토 → 수정 → 심의 제출 → 승인             │
│            (즉시, 자동)   (사전)    (1-2일)   (반려율↓)       │
└─────────────────────────────────────────────────────────────┘
```

### 2.3 도입 효과 요약
- **심의 준비 시간 80% 단축**: AI 사전 검토로 수정 포인트 즉시 파악
- **반려율 50% 감소**: 사전 위반 사항 탐지 및 수정
- **일관된 심의 품질**: AI 기반 객관적 기준 적용
- **최신 법규 자동 반영**: RAG 시스템으로 법규 업데이트 즉시 적용

## 4. 핵심 기능 상세

### 4.1 OCR 텍스트 추출

#### 지원 엔진
| 엔진 | 정확도 | 속도 | 비용 |
|------|--------|------|------|
| Naver Clova OCR | 98.9% | ~1초 | 월 1,000건 무료 |
| PaddleOCR (대안) | ~95% | ~2-3초 | 무료 (오픈소스) |

#### 특징
- 한글/영문 혼합 텍스트 정확 인식
- 이미지 내 다양한 폰트 지원
- 사용자 선택 기반 엔진 전환

### 4.2 위반 키워드 탐지

#### 탐지 카테고리 (7개)
| 카테고리 | 키워드 수 | 예시 | 심각도 |
|----------|----------|------|--------|
| 절대적 표현 | 10개 | 100%, 완벽, 확실 | HIGH |
| 최상급 표현 | 10개 | 최고, 최초, 유일 | HIGH |
| 효능 과장 | 12개 | 완치, 기적, 획기적 | HIGH |
| 비교 광고 | 4개 | 최저가, 1위 | MEDIUM |
| 환자 유인 | 9개 | 무료, 할인, 이벤트 | MEDIUM |
| 의료인 비하 | 3개 | - | LOW |
| 기타 금지 | 5개 | 부작용 없음 | HIGH |

#### 위험도 점수 계산
```
점수 = Σ(키워드 심각도 × 가중치) + 반복 가산점

가중치: HIGH=30점, MEDIUM=15점, LOW=5점
반복 가산: 동일 키워드 2회 이상 시 +5점/회

판정 기준:
- 0점: SAFE (통과)
- 1-19점: LOW (통과 가능)
- 20-49점: MEDIUM (수정 필요)
- 50-99점: HIGH (반려 예상)
- 100점+: CRITICAL (반려 확실)
```

### 4.3 AI 법적 근거 생성

#### GPT 기반 분석
- **모델**: GPT-5.2 (OpenAI Responses API)
- **역할**: 의료법 전문가 페르소나
- **출력**: 위반 사항, 법적 근거, 수정 권고

#### RAG (Retrieval-Augmented Generation)
- **벡터 DB**: Chroma (로컬 영구 저장)
- **임베딩**: OpenAI text-embedding-3-small
- **문서**: 의료법 제56조, 심의 가이드라인
- **검색**: Top-5 유사도 기반

### 4.4 결과 대시보드

#### 제공 정보
| 항목 | 내용 |
|------|------|
| 종합 판정 | 통과/반려 예상 |
| 위험도 점수 | 0-100점 |
| 위반 키워드 | 하이라이팅 표시 |
| 법적 근거 | 관련 법조항 인용 |
| 수정 권고 | AI 생성 개선안 |
| OCR 원문 | 추출된 전체 텍스트 |

# 의료광고 샘플 이미지 수집 가이드

## 📋 수집 목표

총 10개의 의료광고 샘플 이미지 수집
- 피부과 광고: 3개
- 치과 광고: 3개
- 성형외과/한의원 광고: 4개

## 🎯 수집 기준

### 1. 이미지 품질
- **형식**: JPG 또는 PNG
- **크기**: 최소 800x600 픽셀 (OCR 정확도를 위해)
- **용량**: 10MB 이하
- **해상도**: 선명하고 텍스트가 읽기 쉬운 이미지

### 2. 광고 내용
다양한 위험도 레벨의 광고 포함:
- **고위험 (3-4개)**: 위반 키워드 5개 이상 (100%, 최고, 완치, 무료 등)
- **중위험 (3-4개)**: 위반 키워드 2-4개
- **저위험 (2-3개)**: 위반 키워드 0-1개 (정상 광고)

### 3. 텍스트 포함
- 한글 텍스트가 명확하게 보이는 광고
- 과장된 표현이나 금지 키워드가 포함된 광고
- 실제 의료기관 광고 (온라인, 지하철, 버스 정류장 등)

## 🔍 수집 방법

### 방법 1: 웹 검색
```
검색어 예시:
- "피부과 광고"
- "치과 이벤트 광고"
- "성형외과 홍보물"
- "한의원 광고 전단지"
- "의료광고 위반 사례"
```

**추천 검색 사이트:**
- 구글 이미지 검색
- 네이버 이미지 검색
- 의료광고 관련 뉴스 기사
- 지자체 의료광고 심의 사례

### 방법 2: 스크린샷
- 인스타그램/페이스북 의료 광고
- 카카오톡 플러스친구 광고
- 지하철/버스 정류장 광고 촬영
- 의료기관 웹사이트 배너

### 방법 3: AI 생성 (테스트용)
실제 광고를 구하기 어려운 경우:
- 텍스트 에디터로 간단한 광고 이미지 생성
- Canva 등 디자인 툴 사용
- **주의**: 실제 광고와 유사하게 작성

## 🚨 주의사항

1. **저작권**: 테스트 목적으로만 사용
2. **개인정보**: 실제 의료기관 전화번호, 주소 등은 가림 처리 권장
3. **불법 광고**: 실제 위반 광고를 수집하되, 재배포하지 않음
4. **윤리적 고려**: 데모 후 샘플 이미지는 적절히 폐기

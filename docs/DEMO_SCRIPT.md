# 데모 영상 촬영 가이드

## 영상 정보
- **목표 길이**: 2분 (120초)
- **해상도**: 1920x1080 (Full HD) 권장
- **포맷**: MP4

---

## 촬영 전 체크리스트

### 환경 준비
- [ ] 서버 실행 확인
  ```bash
  # Backend
  cd /home/bang/project/MED-AD-REV/backend
  source ../venv/bin/activate
  uvicorn main:app --host 0.0.0.0 --port 8000 --reload

  # Frontend
  cd /home/bang/project/MED-AD-REV/frontend
  npx next dev --hostname 0.0.0.0 --port 5173
  ```
- [ ] 브라우저 탭 정리 (불필요한 탭 닫기)
- [ ] 브라우저 북마크바 숨기기 (Ctrl+Shift+B)
- [ ] 브라우저 확대/축소 100% 확인 (Ctrl+0)
- [ ] 알림 끄기 (방해 금지 모드)
- [ ] 마이크 테스트 (나레이션 녹음 시)

### 샘플 이미지 준비
| 순서 | 파일명 | 용도 | 예상 결과 |
|------|--------|------|-----------|
| 1 | 보톡스.jpg | 저위험 샘플 | LOW (15점) - 통과 |
| 2 | 라식.jpg | 고위험 샘플 | CRITICAL (110점) - 반려 |
| 3 | 성형외과.jpg | 중위험 샘플 | MEDIUM/HIGH |

### 화면 설정
- 브라우저 창 크기: 1920x1080 또는 1600x900
- 주소창에 `http://192.168.0.2:5173` 표시
- 개발자 도구 닫기 (F12)

## 데모 시나리오 (2분)

### Part 1: 시스템 소개 (0:00 - 0:20)
**화면**: 메인 페이지 첫 화면

**나레이션 (20초)**:
> "안녕하세요. 의료광고 AI 심의 시스템을 소개해 드리겠습니다.
> 이 시스템은 의료광고 이미지를 업로드하면, AI가 자동으로 텍스트를 추출하고
> 의료법 위반 여부를 분석하여 심의 통과 가능성을 예측합니다."

**액션**:
- 화면 전체 모습 보여주기
- 마우스로 주요 영역 가리키기 (업로드 영역, 옵션 영역)

---

### Part 2: 이미지 업로드 및 분석 (0:20 - 1:00)
**화면**: 파일 업로드 → 분석 진행 → 결과 표시

**나레이션 (40초)**:
> "먼저 분석할 광고 이미지를 업로드합니다.
> 드래그 앤 드롭 또는 클릭으로 파일을 선택할 수 있습니다."
>
> (파일 선택 후)
> "OCR 엔진은 Naver Clova OCR과 PaddleOCR 중 선택할 수 있고,
> AI 심층 분석과 RAG 모드를 활성화하면 더 정확한 법적 근거를 제공합니다."
>
> (분석 시작 클릭)
> "분석 시작 버튼을 클릭하면 실시간으로 진행률이 표시됩니다."

**액션**:
1. `보톡스.jpg` + `라식.jpg` 2개 파일 선택
2. OCR 엔진: Naver Clova OCR 선택 상태 확인
3. AI 심층 분석: ON
4. RAG 모드: ON
5. "분석 시작" 버튼 클릭
6. 진행률 바 움직이는 모습 촬영

---

### Part 3: 분석 결과 확인 (1:00 - 1:40)
**화면**: 결과 테이블 → 상세 모달

**나레이션 (40초)**:
> "분석이 완료되면 결과 테이블에서 각 파일의 판정 결과를 확인할 수 있습니다.
> 보톡스 광고는 15점으로 '통과 가능' 판정을 받았고,
> 라식 광고는 110점으로 '반려 예상' 판정을 받았습니다."
>
> (상세 버튼 클릭)
> "상세 버튼을 클릭하면 위반 사항, 관련 법규, AI 분석 근거를 확인할 수 있습니다.
> 여기서 '100% 성공', '완치 보장' 같은 금지 표현이 검출되었고,
> 의료법 제56조 위반에 해당합니다."

**액션**:
1. 결과 테이블 전체 모습 보여주기
2. 필터 버튼 클릭해서 필터링 기능 시연 (선택)
3. `라식.jpg`의 "상세" 버튼 클릭
4. 모달에서 위반 사항 목록 스크롤
5. AI 분석 근거 섹션 보여주기
6. OCR 추출 텍스트 섹션 보여주기

---

### Part 4: 관리자 기능 및 마무리 (1:40 - 2:00)
**화면**: Admin 페이지 간략 소개 → 마무리

**나레이션 (20초)**:
> "관리자 페이지에서는 RAG 분석에 사용되는 법규 문서를 관리할 수 있습니다.
> 새로운 가이드라인이나 법규가 추가되면 즉시 분석에 반영됩니다."
>
> "의료광고 AI 심의 시스템은 심의 준비 시간을 단축하고,
> 사전에 위반 요소를 탐지하여 반려율을 낮추는 데 기여합니다.
> 감사합니다."

**액션**:
1. Admin 버튼 클릭하여 관리자 페이지 이동
2. 문서 목록 간략히 보여주기
3. 메인 페이지로 돌아와서 마무리

## 녹화 도구 추천

- **OBS Studio** (추천)
  ```bash
  sudo apt install obs-studio
  ```

  ```bash
  sudo apt install simplescreenrecorder
  ```
- **Kazam**
  ```bash
  sudo apt install kazam
  ```
